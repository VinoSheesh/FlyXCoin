## üìñ About Project

**FlyXCoin** adalah aplikasi pelacak aset kripto berbasis Android Native yang dirancang untuk kecepatan dan akurasi. Aplikasi ini memadukan kekuatan **Jetpack Compose** untuk UI yang responsif dengan **TradingView Lightweight Charts** untuk analisis teknikal tingkat lanjut.

Tujuannya adalah menyediakan pengalaman memantau pasar kripto yang *seamless*, estetik, dan informatif bagi pengguna kasual maupun trader.

## üõ† Tech Stack

FlyXCoin dibangun dengan standar pengembangan Android modern:

* **Language:** [Kotlin](https://kotlinlang.org/) (100%)
* **UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material Design 3)
* **Architecture:** MVVM (Model-View-ViewModel) & Clean Architecture principles.
* **Data Source:** [CoinGecko API](https://www.coingecko.com/en/api)
* **Charting:** [TradingView Lightweight Charts for Android](https://github.com/tradingview/lightweight-charts-android)
* **Dependency Injection:** Hilt
* **Network:** Retrofit & OkHttp
* **Async:** Coroutines & Flow

## üíª Installation

Ingin mencoba menjalankan aplikasi ini di lokal komputer kamu?

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/VinoSheesh/FlyXCoin.git](https://github.com/VinoSheesh/FlyXCoin.git)
    ```
2.  **Open in Android Studio**
    Buka Android Studio, pilih `Open`, dan arahkan ke folder `FlyXCoin`.
3.  **Sync & Run**
    Tunggu sinkronisasi Gradle selesai, lalu jalankan pada Emulator atau Device fisik.
